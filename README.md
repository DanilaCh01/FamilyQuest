# FamilyQuest

FamilyQuest is a gamified cross-platform application designed to help families track behavior and achieve goals. It enables parents to manage tasks and rewards while motivating children through a progress-based system.


## Prerequisites

Before running the project, make sure you have the following installed:
* **Node.js** (v18 or higher recommended)
* **npm** (comes with Node.js)


## Tech Stack

* **Core:** React, Vite
* **Styling:** TailwindCSS (v4)
* **Linting & Formatting:** ESLint, Prettier


## NPM Scripts

* `npm run start:dev` - Starts the development server.
* `npm run build` - Builds the app for production.
* `npm run preview` - Locally preview the production build.
* `npm run lint` - Checks code quality using ESLint.
* `npm run format` - Formats code using Prettier.


## How to Run

### Development Mode
1.  Install dependencies:
    ```bash
    npm install
    ```
2.  Start the dev server:
    ```bash
    npm run start:dev
    ```
3.  Open [http://localhost:5173](http://localhost:5173) to view it in the browser.


### Production Build
1.  Build the project:
    ```bash
    npm run build
    ```
2.  (Optional) Preview the build locally:
    ```bash
    npm run preview
    ```
